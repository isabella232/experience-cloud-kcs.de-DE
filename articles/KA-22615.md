# AEM GraphQl-Konfigurationsprobleme

## Beschreibung {#description}


<b>Beschreibung</b> :



Der Kunde entwickelt eine Website, auf der Daten von seinen AEM mithilfe AEM Headless-Lösung abgerufen werden.

Um den Endpunkt einzurichten, finden Sie im Tutorial unter `[` 1`]`  befolgt wurde.

Nach der Veröffentlichung aller Inhaltsfragmente werden alle Modelle, Inhalte, Verweise, Assets und Abfragen wie in `[` 2`]` Wenn Sie versuchen, ein Inhaltsfragment aus der Veröffentlichungsinstanz abzurufen, wird ein 404-Fehler empfangen, der verhindert, dass der Inhalt vom Herausgeber gelesen wird.
