# Fehler beim Zugriff auf das Profil

## Beschreibung


Fehler beim Zugriff auf das Profil 

BAS-010040 &quot;cusFieldbyTest&quot;(Typ Text) kann nicht in den Integer-Typ (32 Bit) konvertiert werden (bAS-010011 Ungültiger Wert)


## Auflösung


Reparieren Sie die Datenbank mithilfe der Option Datenbank reparieren . Das Problem sollte dadurch behoben werden.
