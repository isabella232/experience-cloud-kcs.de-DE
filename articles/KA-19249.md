# Exportieren lokaler Verweise eines Assets im JSON-Format

## Beschreibung


Sie können die lokalen Verweise eines Assets in [!UICONTROL Verweise] Tab von [!UICONTROL Eigenschaften]. (in [!DNL Assets] Konsole, Asset auswählen, auswählen [!UICONTROL Eigenschaften] in der Symbolleiste und wählen Sie [!UICONTROL Verweise] Registerkarte.) Wenn Sie die Informationen außerhalb von AEM verwenden, können Sie sie im JSON-Format exportieren.

<b>Umgebungen</b>

AEM as a Cloud Service, AEM 6.5


## Auflösung


Beim Zugriff auf die folgende URL &#39;1&#39; werden lokale Verweise im JSON-Format zurückgegeben. `path` -Werte in der Antwort sind die Inhaltspfade, die auf ein Ziel-Asset verweisen.

&#39;1&#39; http://localhost:4502/bin/wcm/references.json?path=path zum Targeting von Assets

<b>Beispielverwendung:</b>
http://localhost:4502/bin/wcm/references.json?path=/content/dam/wknd/en/activities/hiking/equipment_6.jpg
