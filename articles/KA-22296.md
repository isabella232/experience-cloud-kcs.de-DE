# Campaign Analytics-Verbindungsfehler: AnalyticsAPIException: erhalten 403, Verboten

## Beschreibung {#description}


### <b>Umgebung</b>

Vor Ort:

- Adobe Analytics
- Adobe Campaign


### Problem/Symptome

Wenn Sie ein externes Konto erstellen, haben Sie Schwierigkeiten, eine Verbindung zu Adobe Analytics herzustellen. Der folgende Fehler wird angezeigt:


```
Cannot process request for method 'getIMSOrgDetails'. 
Error Message: 'AnalyticsAPIException: received 403,Forbidden, `<` HTML`>` `<` HEAD`>` '. 
Request Details: '{"method":"getIMSOrgDetails","internalName":"cxb_AC_AAIntegration_CXB","useUserAccessTokenIfPresent":"true"}'
```



## Auflösung {#resolution}


Wenn als On-Premise-Kunde eine Firewall auf Ihrer Seite implementiert ist, müssen Sie den Analytics-Endpunkt hinzufügen `analytics.adobe.io/discovery/me` auf Ihre Zulassungsliste.

### Ursache

Der Analytics-Endpunkt war nicht in der Zulassungsliste &quot;&quot;.
