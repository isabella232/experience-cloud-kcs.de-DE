# [!UICONTROL Datei-Wächter] Aktivitätserweiterung in ACS

## Beschreibung

Da wir nicht über die [!UICONTROL Datei-Wächter] -Aktivität in ACS können Sie sie mit den folgenden Aktivitäten erstellen und erhalten mehr oder weniger dieselbe Funktionalität.

## Lösung


![](assets/8e624e34-2cce-ec11-a7b5-0022480a8e40.png)



Die [!UICONTROL Datei-Wächter] ermöglicht das Herunterladen mehrerer Dateien basierend auf dem Dateinamenmuster von einem externen Server. Jede Datei wird an die [!UICONTROL Datei laden] -Aktivität, um Daten in die Aktualisierung aufzunehmen.
Nach dem Abschluss wird die folgende Datei verwendet usw.
Die Dateiübertragung verfügt auch über die [!UICONTROL Verlaufsparameter], wie Sie es aus [!UICONTROL Datei-Wächter].
